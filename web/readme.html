Information générale

Pour ce projet, j'ai voulu inclure un maximum de ce que nous avons étudié cette année en NSI ainsi que des connaissances personnelles que j'ai acquises en programmant depuis un certain temps maintenant. Ce projet inclut donc Python, CSS, HTML et Javascript.
En outre, j'ai voulu inclure une fonctionnalité d'apprentissage automatique dans ce projet, car il s'agit d'un thème particulièrement pertinent à l'heure actuelle.
J'ai donc décidé d'utiliser le module python NEAT pour intégrer des capacités d'apprentissage automatique dans les remakes des jeux rétro que j'ai réalisés en python à l'aide du module pygame.

Les problèmes rencontrés

  • NEAT module en Python
  • Python menu et les différentes aspects interactives à lui
  • Le page web et les éléments interactives aisi que les changements qui dépendent de l'écran de l'utilisateur

Possibilités de développement futur

  • Un espace pour des discusions sur le page web
  • Le programme qui court indefinitivèment sur un Raspberry Pi
  • D'autres jeux video pour montrer la puissance de NEAT
